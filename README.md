# Wonderlust

Wonderlust is an Airbnb-inspired web application where users can explore, create, and review travel listings.

## Tech Stack
- Node.js
- Express.js
- MongoDB
- EJS
- Bootstrap
- Cloudinary
- Passport.js

## Features
- User Authentication
- Create, Edit and Delete Listings
- Image Uploads
- Reviews and Ratings
- Responsive UI
